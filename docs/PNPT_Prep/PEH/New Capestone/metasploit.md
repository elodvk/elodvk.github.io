This guide provides a deep dive into the Metasploit Framework's command-line interface (`msfconsole`) and its premier payload, Meterpreter. We'll move beyond the basics to cover advanced use cases and practical workflows.

## Understanding the Core Components

 - **msfconsole** 🚀: This is your interactive command center. It's where you find, configure, and launch exploits, manage listeners, and handle sessions. It's the "cockpit" of the framework.

 - **meterpreter** 🛠️: This is an advanced, in-memory post-exploitation payload. It runs entirely in the memory of the compromised process, leaving minimal forensic evidence on the hard drive. It's your "multi-tool" for controlling a compromised system.
