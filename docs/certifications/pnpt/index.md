## Overview

The PNPT by TCM Security is a practical penetration testing certification that tests real-world skills. The exam involves performing a full network penetration test and delivering a professional report.

## What It Covers

- **OSINT** — Open Source Intelligence Gathering
- **External Pentest** — Attacking external-facing infrastructure
- **Internal Pentest** — Active Directory attacks, lateral movement
- **Report Writing** — Professional penetration testing reports
