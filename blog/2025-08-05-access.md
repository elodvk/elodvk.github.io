An MDB file is a Microsoft Access database file used to store data, including tables, queries, and forms. It is the default format for Access databases created in versions 2003 and earlier. So, the first thing I am going to do is to try extracting data from `backup.mdb`. We can use `mdb-tables` to list the tables, but since there are many tables inside, I tried to get only the groups with `auth`.

```shell
┌──(elodvk㉿kali)-[~/hack-the-box/access/loot]
└─$ mdb-tables -1 backup.mdb | grep auth   
auth_group_permissions
auth_message
auth_permission
auth_user
auth_user_groups
auth_user_user_permissions
auth_group

```

`auth_user` tables seems interesting, so I extracted the data from it using `mdb-export`:

```shell
┌──(elodvk㉿kali)-[~/hack-the-box/access/loot]
└─$ mdb-export backup.mdb auth_user
id,username,password,Status,last_login,RoleID,Remark
25,"admin","admin",1,"08/23/18 21:11:47",26,
27,"engineer","access4u@security",1,"08/23/18 21:13:36",26,
28,"backup_admin","admin",1,"08/23/18 21:14:02",26,
```

Using the password `access4u@security`, I am able to extract the zip file `Access Control.zip` which has just 1 file called `Access Control.pst`.

Using the `readpst` command, I converted the `.pst` file to `mbox` format.

```shell
┌──(elodvk㉿kali)-[~/hack-the-box/access/loot]
└─$ readpst Access\ Control.pst

Processing Folder "Deleted Items"
	"Access Control" - 2 items done, 0 items skipped.
```
